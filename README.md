# Xclone - A Sleek Twitter/X Clone

A modern, responsive clone of X (formerly Twitter) built with vanilla HTML, CSS, and JavaScript. This project demonstrates clean UI/UX design and interactive web development.

## 🚀 Features

- **Modern Dark Theme**: Sleek dark mode interface matching X's design language
- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile
- **Interactive Posts**: Create new posts, like, retweet, and reply to posts
- **Real-time Updates**: Dynamic post feed with instant interactions
- **Smooth Animations**: Beautiful transitions and hover effects
- **Clean Code**: Well-structured, maintainable vanilla JavaScript

## 🚦 Getting Started

### View Locally

1. Clone the repository:
```bash
git clone https://github.com/rajatkumar1011/Xclone.git
```

2. Navigate to the directory:
```bash
cd Xclone
```

3. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply drag and drop `index.html` into your browser.

## 📱 Responsive Breakpoints

- **Desktop**: Full three-column layout (>1280px)
- **Tablet**: Collapsed sidebar with main content (688px - 1280px)
- **Mobile**: Single column, hidden sidebars (<688px)

## 🎨 Color Scheme

The app uses X's official dark theme colors:
- Background Primary: `#000000`
- Background Secondary: `#16181C`
- Text Primary: `#E7E9EA`
- Text Secondary: `#71767B`
- Accent Blue: `#1D9BF0`

## 📝 Usage

1. **Create a Post**: Type in the "What's happening?!" text area and click "Post"
2. **Like a Post**: Click the heart icon to like/unlike
3. **Retweet**: Click the retweet icon to retweet/unretweet
4. **Reply**: Click the comment icon to reply (focuses input with @mention)
5. **Follow Users**: Click "Follow" on suggested users in the right sidebar
